# ikasoa zk #

## 概述 ##

  ikasoa利用zookeeper注册和更新服务.

## 开发运行环境要求 ##

- 要求java运行环境为java8

## 环境搭建 ##

##### Maven配置 #####

需要修改pom.xml文件,添加依赖:

pom.xml

```xml
    ......
    <dependency>
        <groupId>com.ikasoa</groupId>
        <artifactId>ikasoa-zk</artifactId>
        <version>0.1-BETA</version>
    </dependency>
    ......
```
